From the substances that remain, all the liquid still retained in them is pressed out, using the press if necessary (fig. 1), and then filtering this liquid, which is always cloudy, before adding it to that already extracted; this filtration can be carried out in a funnel over cotton. These operations are repeated several times depending on the case. In the end a mass remains which is generally not so exhausted that it should simply be thrown away. For the most part, in factories of some importance, where it is worth carrying out these manipulations, these residues are subjected to distillation with water; for this purpose they are stored, as they accumulate, in a suitable well-closed cask. By this distillation all the alcohol still contained in them is recovered, mixed with some aromatic principles, so that it can be used in the manufacture of lower-grade liqueurs.

Instead of maceration by simple immersion of the drug in the liquid, one can apply, with much greater benefit, the extraction of the aromatic, bitter, coloring principles, etc. by means of displacement leaching (lisciviazione a spostamento), as is practiced in pharmacies. This procedure is especially useful for the preparation of concentrated extracts (tinctures) for liqueurs, in which the solvent (alcohol) must completely exhaust the drugs.
