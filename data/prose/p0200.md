# Old Foreign Weights and Measures Still in Use — Argentina

The use of alcohol, sugar and essential oils requires careful measurement: an inaccurate evaluation of these ingredients evidently leads either to a financial loss for the manufacturer — if the quantity estimated is greater than what is actually needed — or to fraud against the buyer and consumer in the opposite case.

In general, especially for large-scale production, both solid and liquid substances are weighed; for liquids this is done using calibrated (tared) containers. However, when only small quantities of a liquid are needed — as is the case with essential oils — it is more convenient to measure the volume using graduated vessels. In this case one must bear in mind the difference between a given quantity of liquid measured by weight and the same quantity measured by volume — a difference that depends on the density, or specific gravity, of the liquid in question. Only for water — and for liquids of the same density — do weight and volume coincide. We refer the reader back to what has already been discussed in detail concerning alcohol.

We believe it useful to list here the names of the old weights and capacity measures still in use in the principal foreign countries.

### Argentina (Republic)

The decimal metric system is used; in addition, the following older units remain in use:

**Weights**

| Unit | Equivalence | Value |
|---|---|---|
| Quintal | = 100 Libras | 45.936 kg |
| Arroba | = 25 Libras | 11.485 kg |
| Libra | = 2 Marcos | 460.000 g |
| Marco | = 8 Onzas | 230.000 g |
| Onza | = 16 Adarmes | 28.800 g |
| Adarme | = 36 Granos | 1.800 g |

**Volume Measures**

| Unit | Equivalence | Value |
|---|---|---|
| Pipa catalana | = 4 Cargas = 6 Barriles | 456.000 L |
| Carga | = 1½ Barril = 16 Cortones | 114.000 L |
| Barril | = 4 Canecas = 10⅔ Cortones | 76.000 L |
| Caneca | = 2⅔ Cortones = 8 Frascos | 19.000 L |
| Cortón | = 3 Frascos | 7.125 L |
| Frasco | = 2 Medios = 4 Quartos = 8 Octavos | 2.375 L |
| Medio | = 2 Quartos = 4 Octavos | 1.187 L |
| Quarto | = 2 Octavos | 0.593 L |
| Octavo | — | 0.297 L |

*Note: figures are reproduced from the original table; a few values (marked in the OCR notes) were reconstructed from internal consistency of the conversion chain, since the source scan was partly illegible.*
