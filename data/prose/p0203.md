# England

Although the metric decimal system is officially prescribed, its application still meets considerable resistance among the population, which prefers to use the weights and measures hitherto in use:

### Weights

| English unit | Equivalence | Metric value |
|---|---|---|
| Hundredweight (cwt) | = 4 Quarters = 112 Pounds | = kg. 50.8023 |
| Quarter | = 28 Pounds | = kg. 12.7005 |
| Pound | = 16 Ounces = 256 Drachms | = gr. 4536 |
| Ounce Avoirdupois (1) | | = gr. 28.3495 |
| Ounce Troy or Apothecaries | | = gr. 31.1035 |
| Drachm | = 60 Grains | = gr. 3.8879 |
| Scruple | = 20 Grains | = gr. 1.2959 |
| Grain | | = gr. 0.0648 |

### Measures of Capacity

| English unit | Equivalence | Metric value |
|---|---|---|
| Pipe (Butt) | = 2 Hogsheads = 126 Gallons | = litres 572.9826 |
| Hogshead | = 63 Gallons | = litres 286.2456 |
| Gallon (Imperial G.) | = 4 Quarts = 8 Pints | = litres 4.5459 |
| Quart | = 2 Pints = 8 Gills | = litres 1.1359 |
| Pint | = 20 Fluid Ounces | = litres 0.5682 |
| Fluid Ounce | = 8 Fluid Drachms | = litres 0.0284 |
| Fluid Drachm | = 60 Minims | = litres 0.0035515 |
| Minim | | = c.c. 0.0592 |

To quickly convert these weights and measures into those of the metric system, proceed as follows:

| Operation | Result |
|---|---|
| Pounds × 0.4536 | = kilograms |
| Pounds (Av.) : 2.2046 | = kilograms |
| Ounces (Av.) × 28.3495 | = grams |
| Ounces (Av.) : 0.0353 | = grams |
| Ounces (Troy) × 31.1035 | = grams |
| Ounces (Troy) : 0.0311 | = grams |
| Drachms × 3.8879 | = grams |
| Scruples × 1.2959 | = grams |
| Grains × 0.0648 | = grams |
| Grains : 15.4324 | = grams |
| Gallons × 4.5459 | = litres |
| Gallons : 0.2199 | = litres |
| Pints × 0.5682 | = litres |
| Pints : 1.7598 | = litres |

(1) Avoirdupois weights are those adopted by the British Pharmacopoeia; Troy weights are those adopted in the trade of precious metals.
